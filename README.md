# Quasar App

> A Quasar project

## Installing Quasar

``` bash
# install quasar-cli globally (sudo required to install globally)
$ sudo npm install -g quasar-cli
```

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
